# Liste chainée
Une liste chainée est une structure composée d'une donnée et un address à un autre type de la même type.  
ex. var = ["A" , "path1"] - path1 = ["B" , "path2"] ... pathn = ["Z" , None]

Les opérations sont les mêmes d'une liste:  
- add
- insert
- remove
- find
- index
  
Avantage:  
- insertion et supression rapide des données
Inconvénient:  
- Les données sont difficile à accéder.
- Utilisation de mémoire.
